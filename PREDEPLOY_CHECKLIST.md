# Predeploy Checklist

Run these before release:

1. `npm run -s lint:deploy`
2. `npm run -s smoke`
3. `npm run -s build`

Manual sanity checks:

1. Run code in editor and confirm output appears in console.
2. Open a challenge and run tests.
3. Sign in and open profile.
4. Create/join collaboration room.
5. Send a room chat message.
6. Open whiteboard and verify snapshot/cursor sync.
